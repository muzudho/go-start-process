# go-start-process

Go言語の練習用（＾～＾）  

外部プロセスを起動する（＾～＾）  

## Set up

```console
go build
```

## Run

```shell
go-start-process -FilePath <ExecutableFilePath> -ArgumentList <CommandLineParameters>
# Example
# go-start-process -FilePath C:/Users/むずでょ/go/src/github.com/muzudho/go-count-up/go-count-up.exe
```
